Applications developpées pour des clients

IZEManager

IZEManager est une application web de gestion d’hotel. L’application permet de gérer les réservations de chambre, attribuer les chambres aux clients, verifier la disponibilité de chambre, générer une facture aux clients, gérer la consommation au restaurant de l’hôtel. Elle est développée en asp .net core pour l’API avec blazor pour le front et SQL Server comme SGBD
